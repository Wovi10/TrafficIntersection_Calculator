# Technical rules for project
## Time
### Input
- Always in s.
- Always Double.
### Usage
- Always in s.
- Always Double.
### Basics
- Abbreviation is 't', always as prefix.
- t = d / t
### Output
- Always in s.
- Always Double.

## Speed
### Input
- Initial in km/h, then always m/s.
- Always Double.
### Usage
- Always in m/s.
- Always Double.
### Basics
- Abbreviation is 'v', always as prefix.
- v = d * (t<sub>exit</sub> - t<sub>entry</sub>)

## Distance
### Input
- Always in m.
- Always Double.
### Usage
- Always in m.
- Always in Double.
### Basics
- Abbreviation is 'd', always as prefix.
- d = v * t
